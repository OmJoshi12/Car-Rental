# Quick Deployment Guide - InfinityFree

## Your Project is Ready for Deployment!

### Step 1: Sign Up (2 minutes)
1. Go to: https://infinityfree.com/
2. Click "Sign Up"
3. Fill in your details (email, password)
4. Verify your email

### Step 2: Create Website (1 minute)
1. After login, click "Create Account"
2. Choose a subdomain (e.g., `omcarrental.infinityfreeapp.com`)
3. Click "Create Account"

### Step 3: Upload Files (3 minutes)
1. Go to your control panel
2. Click "File Manager"
3. Delete default files (index.html, etc.)
4. Upload ALL files from your project folder:
   - Upload entire project folder contents
   - Make sure `setup.php` is uploaded

### Step 4: Setup Database (2 minutes)
1. In control panel, go to "MySQL Database"
2. Note your database credentials:
   - Database name
   - Username  
   - Password
   - Hostname

### Step 5: Run Installation (1 minute)
1. Visit: `your-domain.com/setup.php`
2. Enter database credentials from Step 4
3. Click "Install Database"
4. Done! Your site is live
